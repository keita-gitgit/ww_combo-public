# 鳴潮コンボノート (ww_combo)

鳴潮（Wuthering Waves）のコンボ・ローテーションをスマートフォンでポチポチ記録するためのツール。

手書きノートの「キャラ名：技 / 技 / 技」形式をそのままデジタル化したもので、
PS5でプレイしながらスマートフォンでローテーションを作成・確認できる。

## 機能

- **ローテーション作成**: 1〜3人のキャラを選び、キャラごとの技パレットから順番に技を追加
- **シンプルな技パレット**: 全キャラ共通の基本技16項目を収録。段数・強化状態・特殊派生は、自分の呼び方で自由に追加・改名・並べ替えできる
- **自由な編集**: 公式技を土台に、技名の変更や自分用の技・呼び方を追加可能。ローテーションと各行の複製、編成変更にも対応
- **一覧整理**: ローテーション名・キャラ名での検索と、お気に入りによる絞り込みに対応
- **並べ替え**: ローテーション、キャラごとの行、行内の技、技パレットを長押しで並べ替え
- **コマンド表示**: 設定したPS5ボタン表記と技名を、プレイ中に見やすい形式で表示
- **参考動画**: YouTubeやXなど、ローテーション作成時に参照したURLを保存
- **バックアップ**: JSONエクスポート / インポート（データは端末のブラウザ内に保存）

## キャラクターデータ

`src/characterData.ts` に実装済み全キャラクター（2026-07-12 / Ver3.5前半 時点、55体）のマスタ、
`src/characterActions.ts` で全55体に共通の基本技16項目を管理している。旧版の詳細技名は、保存済みコンボを壊さず基本技へ移行するためにのみ保持している。
アイコン画像（`public/chars/`）はゲーム内の公式素材（© Kuro Games）で、個人利用目的で同梱している。
新キャラ追加時はキャラ情報とアイコンを追加すると、共通の基本技が既存の保存データにも起動時に自動で反映される。
同期時はユーザーが追加・改名した技や、既存コンボから参照されている技を維持する。

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 型チェック + ビルド（dist/ に出力）
npm run preview  # ビルド結果の確認
```

技術構成: Vite + React + TypeScript / localStorage 保存 / サーバー不要の静的Webアプリ

## プライバシー

- 入力したデータを外部サーバーへ送信する処理や、アクセス解析・外部スクリプトは使用していない
- データは利用中のブラウザの `localStorage` にのみ保存される
- エクスポートしたJSONにはパーティ名・ローテーション名・メモが含まれるため、公開リポジトリやSNSへ共有しない
- 標準のバックアップファイル名（`ww-combo-backup-*.json`）はGitの追跡対象から除外している

## PWA・GitHub Pages

- スマートフォンのホーム画面へ追加可能
- 初回読み込み後はオフラインでも起動可能
- 新しいバージョンを検出すると自動的に更新
- `main` ブランチへのpushでGitHub Actionsがビルドし、GitHub Pagesへ自動公開

初回公開時は、リポジトリをパブリックにしたうえで、GitHubの
`Settings > Pages > Build and deployment > Source` を `GitHub Actions` に設定する。
以後は `main` の更新だけで同じURLへ最新版が公開される。

設計の経緯は [docs/research.md](docs/research.md) を参照。
